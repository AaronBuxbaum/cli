## `percy build:finalize`

Finalize parallel Percy builds where PERCY_PARALLEL_TOTAL=-1

```
USAGE
  $ percy build:finalize

OPTIONS
  -q, --quiet    log errors only
  -v, --verbose  log everything
  --silent       log nothing

EXAMPLE
  $ percy finalize
```
